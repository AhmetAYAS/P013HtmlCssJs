Githup Notları

Temel Terimler

  1. repository : Kısa ismiyle repo, kodlarımızın saklandığı yerdir
  2. master : Depodaki ana proje sürümüdür
  3. branch : Git ağaç mantığı ile çalışır ve projedeki geliştirmelerimizi ağaç dalı mantığı ile ayrı dallarda saklayabilir, sonrasında master branch de birleştirebiliriz
  4. merge : branch lerde yapılan işlemlerin master da birleştirilmesidir.
  5. commit : commit işlemi lokaldeki dosyalarımızı depoya gönderilmek üzere hazırlar
  6. push : dosyalarımızı depoya göndermemizi sağlar
  7. pull : Depodaki dosyalarımızı çekmemizi sağlar
  8. .gitignore : Git in takip etmesini istediğimiz dosya ve klasörleri belirttiğimiz dosyadır.

Vs code da projede bir değişiklik yapıldıktan sonra bunu githup a gönderilmek için source control tabını açtık. message alanına yapılan değişiklikle alakalı bir mesaj yazdık. Son olarak Commit butonuna basıp depoya gönderilmek üzere hazırladık ve Sync butonuna basarak push işlemini yaparak değişiklii githup a yolladık.

Temel Git Komutları

  1. git add veya git add * git add index.html : Klasöre eklenen dosyaları gönderilmek üzere sahneye alma (Stage)
  2. git status : klasördeki tüm değişiklikleri göster
  3. git commit -m "ilk commit" : Depoya gönderilmek için commit işlemi yapılır
  4. git commit -a -m "Mesajımız : Staging e ekle ve commitle
  5. git push : değişiklikleri depoya gönderme komutu
  6. git pull : Uzak depodaki dosyaları lokale indirme

Githup a yüklediğimiz projenin gizlilik ayarını githup daki proje içerisine girip > settings den ayarları açıp > en alta indik orada change visibility butonuna basarak private veya public yapabiliriz.

Githup dan projeyi silmek için yine settings deki danger zone alanındaki delete this repository butonuna basarak gelen pencerefen bizden istediği yazıyı kutucuğa girerek silmeyi onaylayıp repoyu silebiliriz

Bir projeyi bir ekiple birleştirmek istersek sttings deki sol menuden collaborators sekmesinden giriş yaparak açılan ekrandan diğer githup kullanıcılarını projeye dahil ederek ortak geliştirme yapabiliriz.
